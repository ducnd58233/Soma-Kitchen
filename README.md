# **When download this project**
- Right click `package.json`
- Click `Open in Integrated Terminal`
- In terminal, write: `npm install`

# **To run this project**
- Right click `package.json`
- Click `Open in Integrated Terminal`
- In terminal, write: `npm run dev`
